# Dr. Pitagoras | Painel Receita Certa

Cockpit empresarial e eleitoral para organizar a operação do movimento Receita Certa, acompanhar conteúdo, território, provas, metas e inteligência da campanha de 2026.

## Rodar localmente

```bash
npm install
npm run dev
```

Abra `http://localhost:3000`.

## Validação

```bash
npm run check
```

## Princípios

- Dados eleitorais e realizações precisam de fonte primária antes da publicação.
- A base de contatos deve operar por consentimento e em conformidade com LGPD e regras eleitorais.
- Números exibidos no MVP são metas estratégicas ou dados de demonstração quando identificados como tal.
